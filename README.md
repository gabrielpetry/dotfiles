

# Dot Files

My dot files are here, i'm adjusting and uploading them as I go

## How does this work?
The management is done via [stow](https://www.gnu.org/software/stow/), 
make work just cd into dotFiles and

```bash
stow *
```
Or you can stow individually

```bash
show zshrc
```

programs\_pacman.csv is the list of programs that I use

programs\_aur.csv is the list of programs that I use that are from AUR


## How to get Started?

```bash
git clone https://github.com/gabrielpetry/petryfiles.git $HOME/petryfiles
```

# Shell in use

The current shell i'm using is zsh with Oh-my-zsh

## Vim Plugins

Need to install [VimPlug](https://github.com/junegunn/vim-plug)

then run :PlugInstall for installing all the plugins
